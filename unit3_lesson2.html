גזרת הגרוניים

פעלים שבשורשם אחת מאותיות א ה ח ע (ולעיתים ר׳) — שינויי ניקוד הנובעים מתכונות הגרוניים

⏱️ 45 דקותאהח״ע + ר׳חטף-פתח / סגולדגש חסר

א מיהם הגרוניים?

אותיות הגרון: אהחע ולעיתים גם ר. אינן יכולות לקבל דגש חזק — גורם לשינויים בניקוד בכל שלושת מיקומי הגרונית בשורש.
  • 🚫
    אי-קבלת דגש כאשר הדגש נדרש — התנועה לפניו מתארכת בפיצוי (תשלום דגש)
  • 🔄
    שווא נע → חטף גרונית אינה נושאת שווא נע פשוט — הוא הופך לחטף-פתח (אֲ) או חטף-סגול (אֱ)
  • 💛
    העדפת פתח גרוניים מושכות את התנועה הסמוכה לכיוון פתח

ב מיקום הגרונית בשורש

מיקוםשם הגזרהדוגמהשורשהשינוי
פ׳פ׳ גרוניתעָמַד → יַעֲמֹדע-מ-דשווא → חטף-פתח
פ׳פ׳ גרוניתאָהַב → יֶאֱהַבא-ה-בשווא → חטף-סגול

עיקרון הפיצוי: כשגרונית אינה יכולה לקבל דגש — התנועה שלפניה מתארכת: קמץ במקום פתח | צרה במקום חירק. זהו "דגש חסר" בלשון המסורת.

ג חטף-פתח לעומת חטף-סגול

כלל הבחירה: הגרונית בוחרת חטף לפי התנועה הדומינטית בצורה:
— רוב פתח/קמץ → חטף-פתח (אֲ עֲ חֲ הֲ)
— רוב סגול/צרה → חטף-סגול (אֱ עֱ)
יַעֲמֹד ← פתח → חטף-פתח
יֶאֱהַב ← סגול → חטף-סגול
נֶעֱזַב ← סגול (נפעל) → חטף-סגול
📌 שלוש התכונות המרכזיות
1. דגש חסר — א ה ח ע ר אינן מקבלות דגש חזק; תנועה לפניהן מתארכת בפיצוי
2. חטף במקום שווא — שווא נע → חטף-פתח (אֲ) או חטף-סגול (אֱ)
3. פתח גנובה — לפני ח׳ / ע׳ / ה׳ / א׳ בסוף מילה
התקדמות: 0 / 8
חלק א׳זיהוי מיקום הגרונית
