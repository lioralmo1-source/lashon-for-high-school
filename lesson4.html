שם המספר

1
זכר ונקבה — הסכמת המספר עם שם העצם

בעברית, המספר חייב להסכים עם שם העצם שהוא מונה — במין (זכר/נקבה). כלל אצבע: שמות עצם זכריים מקבלים מספר נקבי, ושמות עצם נקביים מקבלים מספר זכרי. זהו ה"כלל ההפוך" המפורסם.

כיצד קובעים את מין שם העצם?

  • הוסיפו את התואר "גדול / גדולה" לפני שם העצם — האם מתאים זכר או נקבה?
  • לחלופין: הוסיפו "יפה" — "ילד יפה" (זכר) / "ילדה יפה" (נקבה)
  • שמות העצם המסתיימים ב-ה/ת בדרך כלל נקביים: כיתה, מחברת, מנורה

דוגמאות להסכמה נכונה:

  • שלושה ילדים (ילד = זכר ← מספר נקבי: שלושה) ✓
  • שלוש ילדות (ילדה = נקבה ← מספר זכרי: שלוש) ✓
  • חמישה כיסאות (כיסא = זכר ← חמישה) ✓
  • חמש מחברות (מחברת = נקבה ← חמש) ✓

2
מספרים מיוחדים — 1 ו-2

המספרים 1 ו-2 נוהגים שונה מכל שאר המספרים — הם מסכימים עם שם העצם (לא הפוך!), ויש להם צורות מיוחדות.

המספר 1 — אֶחָד / אַחַת:

  • עם שם עצם זכרי: יֶלֶד אֶחָד, כיסא אֶחָד (המספר בא אחרי שם העצם!)
  • עם שם עצם נקבי: יַלְדָּה אַחַת, מחברת אַחַת

המספר 2 — שְׁנַיִם / שְׁתַּיִם:

  • עם שם עצם זכרי: שְׁנֵי ילדים / שְׁנַיִם (עצמאי)
  • עם שם עצם נקבי: שְׁתֵּי ילדות / שְׁתַּיִם (עצמאי)

הצורה בסמיכות: שְׁנֵי כיסאות, שְׁתֵּי מחברות.

⚠️ שגיאה נפוצה מאוד: "יש לי שתי שקלים" — שגוי! שקל הוא זכר, לכן: "יש לי שני שקלים". "שתיים" ו-"שניים" הן הצורות העצמאיות (ללא שם עצם): "יש לי שתיים" (= שתי ידיים, במשמעות עצמאית).
3
תאריכים — תמיד בזכר

בעברית, כאשר מציינים תאריך בחודש (יום בחודש), המספר תמיד בצורת הזכר — גם אם חודש הוא זכרי וגם אם הוא נקבי.

דוגמאות נכונות:

  • חֲמִשָּׁה בְּמַאי (לא: חמש במאי)
  • שְׁלוֹשָׁה בְּיוּנִי (לא: שלוש ביוני)
  • עֶשְׂרִים וְאַרְבָּעָה בְּנוֹבֶמְבֶּר
  • הַשְּׁמִינִי בְּמַרְץ (מספר סידורי — גם הוא בזכר)

הסיבה: "יום" הוא שם עצם זכרי — "יום חמישה בחודש" — ולכן המספר נקבי (=חמישה).

💡 טריק לזכירה: "היום הוא חמישה במאי" — המילה הנסתרת היא "יום" (זכר) ← מספר נקבי = חמישה. תמיד שאלו בשקט: "כמה היום?"
4
גילאים — תמיד בנקבה

ביטויי גיל בעברית — "בת X", "בן X" — המספר בא תמיד בצורת הנקבה, ללא קשר למין האדם.

דוגמאות:

  • הוא בֶּן שֵׁשׁ (לא: בן ששה)
  • היא בַּת שֶׁבַע (לא: בת שבעה)
  • הם בני שֵׁשׁ עֶשְׂרֵה (לא: ששה עשר)
  • היא בת עֶשְׂרִים וְשָׁלוֹשׁ (לא: עשרים ושלושה)

הסיבה: "שנה" הוא שם עצם נקבי — "בן שש שנים" — ולכן המספר זכרי (= שש). הכלל ההפוך!

⚠️ שגיאה נפוצה: "הוא בן שישה" — שגוי! "שנה" היא נקבה → מספר זכרי → "בן שֵׁשׁ". מבחן: הוסיפו "שנים" — "בן שש שנים" ← שנה נקבה ← מספר זכרי = שש.
5
המספר המעורב — 11 עד 19 ו-21 ואילך

מספרים מעורבים (הכוללים יחידות) מחייבים הסכמה כפולה: גם העשרות גם היחידות מסכימות עם שם העצם.

מספרים 11–19 (עשר + יחידה):

  • זכר: אַחַד עָשָׂר ילדים | שְׁנֵים עָשָׂר חודשים | שְׁלוֹשָׁה עָשָׂר שבועות
  • נקבה: אַחַת עֶשְׂרֵה ילדות | שְׁתֵּים עֶשְׂרֵה תלמידות | שְׁלוֹשׁ עֶשְׂרֵה מדינות

מספרים 21 ואילך:

  • 21 ילדים: עֶשְׂרִים וְאֶחָד (היחידה מסכימה — ילד זכר ← אחד)
  • 21 ילדות: עֶשְׂרִים וְאַחַת (היחידה מסכימה — ילדה נקבה ← אחת)
  • 23 ספרים: עֶשְׂרִים וּשְׁלוֹשָׁה (ספר זכר ← שלושה)
  • 23 כיתות: עֶשְׂרִים וְשָׁלוֹשׁ (כיתה נקבה ← שלוש)

כלל: העשרות (עשרים, שלושים...) אינן משתנות. רק היחידות מסכימות עם שם העצם.

💡 שיטת העבודה: (1) קבעו את מין שם העצם. (2) הסכימו רק את היחידה. (3) העשרה נשארת ללא שינוי.
דוגמה: 32 שאלות → שאלה = נקבה → היחידה (2) = שְׁתַּיִם → שְׁלוֹשִׁים וּשְׁתַּיִם שאלות ✓

פתרונות — לשימוש מורה בלבד

שיעור 4 — שם המספר

א חלק א׳ — התאמת מספר לשם עצם (שאלות 1–5)
1 כתיבת המספר הנכון
🔑 מפתח — שאלה 1 א. חֲמִשָּׁה כיסאות — כיסא = זכר ← מספר נקבי (ה-צורה)
ב. שְׁמוֹנֶה מחברות — מחברת = נקבה ← מספר זכרי (הצורה הקצרה)
ג. שְׁלוֹשָׁה ספרים — ספר = זכר ← מספר נקבי
כלל: זכרים + נקבי | נקביים + זכרי — "הכלל ההפוך".
2 בחירת הצורה הנכונה
🔑 מפתח — שאלה 2 א. עשרים וחמישה תלמידים — תלמיד = זכר ← חמישה (נקבי)
ב. ארבע עגבניות — עגבנייה = נקבה ← ארבע (זכרי)
ג. שישה פרקים — פרק = זכר ← שישה (נקבי)
3 המספרים 1 ו-2
🔑 מפתח — שאלה 3 א. שני שקלים — שקל = זכר ← שְׁנֵי (סמיכות זכר)
ב. שתי לחמניות — לחמניה = נקבה ← שְׁתֵּי (סמיכות נקבה)
ג. סרט אחד — סרט = זכר ← אֶחָד (מסכים, בא אחרי שם העצם)
1 מסכים (לא הפוך!): זכר=אחד, נקבה=אחת. שני/שתי בסמיכות; שניים/שתיים עצמאי.
4 מספר מעורב — 21 שאלות
🔑 תשובה נכונה — ב׳: עשרים ואחת שאלה = נקבה ← היחידה (1) = אַחַת (זכרי). עשרים אינה משתנה.
סדר נכון: עשרות לפני יחידות. "אחת ועשרים" = שגוי בסדר.
5 כתיבת מספרים מעורבים
🔑 מפתח — שאלה 5 א. שְׁלוֹשָׁה עָשָׂר תלמידים (תלמיד = זכר, 13 = שלושה עשר)
ב. שְׁבַע עֶשְׂרֵה מדינות (מדינה = נקבה, 17 = שבע עשרה)
ג. שְׁלוֹשִׁים וּשְׁתַּיִם שאלות (שאלה = נקבה, 32 = שלושים ושתיים)
ד. אַרְבָּעִים וְחָמֵשׁ דקות (דקה = נקבה, 45 = ארבעים וחמש)
ב חלק ב׳ — תאריכים וגילאים (שאלות 6–10)
6 כתיבת תאריכים
🔑 מפתח — שאלה 6 א. שְׁלוֹשָׁה במרץ — תאריך = יום (זכר) ← מספר נקבי
ב. שְׁמוֹנָה בספטמבר — תאריך תמיד בזכר
ג. חֲמִשָּׁה עָשָׂר ביוני — 15 = חמישה עשר (זכר)
זכרו: "כמה היום?" ← יום = זכר ← מספר נקבי.
7 כתיבת גילאים
🔑 מפתח — שאלה 7 א. בן שֵׁשׁ — גיל = שנה (נקבה) ← מספר זכרי
ב. בת שֵׁשׁ עֶשְׂרֵה — 16 בנקבה (שש + עשרה)
ג. בני עֶשְׂרִים וְשָׁלוֹשׁ — 23 בנקבה (עשרים + שלוש)
מבחן: "בן כמה שנים?" ← שנה נקבה ← מספר זכרי.
8 תאריך — "שבעה באוקטובר"
🔑 תשובה נכונה — ג׳: שבעה באוקטובר 7 = שִׁבְעָה (מספר נקבי = לשמות זכריים). "יום" = זכר ← שבעה.
"שביעי" = מספר סידורי (ראשון, שני, שלישי...) — שגוי בהקשר זה.
9 משפטים — תאריך וגיל
🔑 מפתח — שאלה 9 א. "נולד בְּאַרְבָּעָה בְּפֶבְרוּאָר" — תאריך ← ארבעה
ב. "היא בת אַרְבַּע עֶשְׂרֵה" — גיל ← ארבע עשרה
ג. "המפגש הוא בְּעֶשְׂרִים בְּיַנוּאָר" — עשרות אינן משתנות
10 מספרים בהקשר מורכב
🔑 מפתח — שאלה 10 א. שְׁתֵּים עֶשְׂרֵה שנים — שנה = נקבה ← שתיים עשרה
ב. שְׁלוֹשִׁים וְאַרְבַּע משפחות — משפחה = נקבה ← ארבע
ג. גיל שְׁמוֹנֶה עֶשְׂרֵה — גיל = נקבה ← שמונה עשרה
ג חלק ג׳ — תיקון שגיאות נפוצות (שאלות 11–15)
11 תיקון שני / שתי
🔑 מפתח — שאלה 11 א. "שתי שקלים" → "שני שקלים" — שקל = זכר ← שְׁנֵי
ב. "שני ערים" → "שתי ערים" — עיר = נקבה ← שְׁתֵּי
ג. "שניים עטים" → "שני עטים" — עט = זכר ← שְׁנֵי (שניים = עצמאי בלבד, לא בסמיכות)
12 תיקון גיל ותאריך
🔑 מפתח — שאלה 12 א. "בן שישה" → "בן שֵׁשׁ" — גיל = נקבה ← שש (זכרי)
ב. "שלוש בינואר" → "שְׁלוֹשָׁה בינואר" — תאריך = זכר ← שלושה (נקבי)
ג. "שבעה עשר" → "שְׁבַע עֶשְׂרֵה" — גיל = נקבה ← שבע עשרה (זכרי)
13 בחירה מרובה — ארבעה תפוחות
🔑 תשובה נכונה — א׳: ארבעה תפוחות תפוח = זכר ← מספר נקבי = אַרְבָּעָה. "ארבע תפוחות" שגוי — ארבע לנקביים.
14 תיקון שגיאות בקטע
🔑 מפתח מלא — 4 שגיאות 1. "שלוש במאי" → שלושה במאי (תאריך = זכר)
2. "שתיים מורות" → שתי מורות (בסמיכות + מורה נקבה)
3. "שלושים ושלושה" (גיל) → שלושים ושלוש (גיל = נקבה)
4. "שישה שנות" → שש שנות (שנה = נקבה ← זכרי)
15 שאלת חשיבה — כתיבה חופשית
🔑 נקודות בדיקה 1. תלמיד/ה = זכר/נקבה → הסכמה הפוכה בהתאם
2. תאריך = זכר ← מספר נקבי
3. גיל = נקבה ← מספר זכרי
4. אחים = זכר; אחיות = נקבה → הסכמה הפוכה לכל אחד
3 נקודות לכל משפט: תוכן (1) + הסכמה נכונה (1) + ניסוח (1).
